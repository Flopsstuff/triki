Getting started

  1. Wake the token — press the button on the cap so it starts advertising.
  2. Click Connect above (and allow Bluetooth when the browser or the OS asks for access).
  3. Pick your TRIKI … device in the chooser popup — streaming then starts automatically.

Raw frame log 






  Lay the token flat → the Z accel axis should read ≈ +1.00 g. Yaw drifts over
  time (6-axis fusion, no magnetometer) — use Reset to re-zero it.
  Switch the fusion filter between Madgwick and VQF (or Accel only
  for gravity-only tilt — instant but jittery, no yaw) and tune its responsiveness with
  the slider.
  Drag the model to orbit the camera; pick the 3D model with the dropdown.
  The rate button steps the IMU sample rate 0.25x (26 Hz) → 4x (416 Hz);
  1x = 104 Hz. Above ~208 Hz BLE may not keep up — watch the actual streaming NNN Hz.
  Only one BLE central can hold the token at a time: close any running bleak script first.
