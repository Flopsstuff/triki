TrikiŻabka Triki as a BLE motion controller

Notes and tooling for the Żabka "Triki" — a collectible BLE token (nRF52810 + LSM6DSL IMU) shaped like a bottle cap. The goal: read its accelerometer/gyroscope over BLE and reuse it as a motion controller.

Hardware

nRF52810 BLE SoC, LSM6DSL accelerometer/gyroscope, and the rest of the board.

BLE protocol

The Nordic UART Service GATT map, the green-LED control register, and the command interface.

IMU streaming

Live accelerometer + gyroscope over BLE via an 8-byte start command and 14-byte motion frames.

triki-controller (npm)

A dependency-free TypeScript package — connect over Web Bluetooth, stream the IMU, and read fused Madgwick orientation in your own app.
